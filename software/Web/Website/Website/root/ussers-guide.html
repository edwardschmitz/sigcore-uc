D88A42 User's Guide
Installation, configuration, and operation guide for hardware and software

Introduction

This guide provides comprehensive information to help you install, configure, and operate both the hardware and software components of the D88A42. The device delivers reliable data acquisition and control solutions for industrial automation, process control, and laboratory measurement environments.

Who Should Use This Documentation?

  • System Integrators: Setting up hardware for field applications.
  • Engineers and Technicians: Installing and configuring the system.
  • Software Developers: Using the C# and Python libraries for custom solutions.
  • End Users: Operating the system through the provided software interface.

How This Documentation Is Organized

Hardware Documentation: Connections, relay usage, and safety considerations.

Software Documentation: Using the control panel, device settings, and channel operations.

Library Documentation: Programming with .NET and Python libraries, examples, and API references.

Warnings and Safety Notice

Read these warnings carefully before installing or operating the system. Failure to comply may result in injury, equipment damage, or voiding of warranties.

General Safety

  • Only qualified personnel should perform installation and maintenance.
  • Always disconnect power before servicing.
  • Never touch exposed terminals when the system is powered.
  • Ensure proper grounding for safe operation.

Electrical Safety

  • Relays may carry up to 120VAC or 30VDC—use caution.
  • Follow proper wiring practices with rated wires and connectors.
  • Do not exceed relay contact ratings (120VAC @ 10A, 30VDC @ 2A).

Environment

  • Indoor, dry environments only.
  • Operating range: 0°C to 50°C.
  • Avoid moisture, flammable atmospheres, and overheating.

Front Panel

The front panel provides clear labeling and organized sections for easy access to I/O connections, power, and relays.

Sections

  • Relay Outputs (R1–R8): SPDT relays rated 120VAC/10A or 30VDC/2A.
  • Digital Inputs (DI1–DI8): 0–0.5V = OFF, 3–5V = ON.
  • Analog Inputs (AI1–AI4): 0–10V standard; jumper configurable for 0–40V.
  • Analog Outputs (AO1–AO2): 0–5V outputs with calibration scaling.
  • 5V Supply: For sensors/logic-level devices (observe current limit).
  • Ground Terminal: Common reference for all I/O.

Relay Usage

Each relay is SPDT, offering NO and NC contacts for flexible control of external loads.

Relay Ratings

  • 120VAC @ 10A
  • 30VDC @ 2A
  • ~10ms switching time
  • Isolated from internal circuitry

Example Connections

Using the internal 5VDC supply for relay activation.

Digital Input Usage

DI1–DI8 accept external voltages to monitor switches, sensors, or relays.

  • OFF: 0–0.5V
  • ON: 3–5V
  • Inputs share a common reference

Analog Input Usage

  • 0–10V standard input range
  • 0–40V extended range with jumper
  • Software calibration and filtering included

Example: reading 0–10V signal on AI1.

Analog Output Usage

AO1 and AO2 provide calibrated 0–5V outputs for control or simulation.

Applications

  • Control proportional valves or drives
  • Provide setpoints to controllers
  • Simulate sensor outputs
